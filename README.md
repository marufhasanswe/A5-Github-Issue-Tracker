# A5-Github-Issue-Tracker

1. What is the difference between var, let, and const? <br>
   Answer: var -> Declares variable with global scope and allows re-declaration and updates within the same scope.<br> let -> Declares variable with block scope, allowing update but not allowing re-declaration.<br> const -> Declare block scope variable that cannot be reassigned after their initial assignment. <br>
2. What is the spread operator (...)? <br>
   Answer: Spread operator is used to expand elements of an object or array.
3. What is the difference between map(), filter(), and forEach()? <br>
   Answer: map() - map is used to transform or modify each element of an array. <br>
   filter() - filter is used to select element of an array that satisfy a condition. <br>
   find() - find is used to get first element of an array that satisfy a condition. <br>
4. What is an arrow function? <br>
   Answer: Arrow function in javascript a shorter and more concise way to write a function using the => syntax. <br>
5. What are template literals? <br>
   Answer: Template literals in javascript are way to create string using backtick `` allowing multi-line strings, embedded expression using ${} this.
